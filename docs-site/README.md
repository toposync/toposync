# Toposync Documentation Site

Docusaurus site for the public Toposync documentation.

## Development

Run from the repository root:

```bash
npm run docs:start
```

The local documentation server uses `http://127.0.0.1:18763/`.

For Portuguese:

```bash
npm run docs:start:pt-BR
```

Docusaurus serves one locale at a time in development. Use `docs:start` for
English and `docs:start:pt-BR` for Portuguese.

## Build

```bash
npm run docs:build
```
