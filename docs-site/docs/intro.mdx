# Toposync documentation

Toposync is a local-first platform for home automation, cameras, spatial context, and extensible processing.

## What is here

- Installation paths for Python, Docker, Home Assistant, and processing servers.
- Product documentation for the core platform, extensions, and future app workflows.
- Developer documentation for integrations and extension authors.

## Current status

This documentation site is being initialized. The existing installation guides remain in the repository while the public documentation structure is prepared.
