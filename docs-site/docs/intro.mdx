# Toposync documentation

Toposync is an open source, local-first platform for **Spatial Home Automation with local intelligence**: cameras, spatial camera mapping, spatial events, 2D/3D home compositions, Home Assistant integration, and extensible processing in private home environments.

The short version:

- **Spatial Camera Mapping** connects camera images to the mapped home.
- **Spatial Events** add position, area, camera, object state, and context to events.
- **Spatial Intelligence** is the broader local interpretation layer that combines cameras, areas, devices, pipelines, and history.
- **Spatial Awareness** is used for area-aware product behavior, such as reacting differently near gates, pools, driveways, sidewalks, or entrances.

## What is here

- Installation paths for Python, Docker, Home Assistant OS, and processing servers.
- Product documentation for the core platform, extensions, and future app workflows.
- Developer documentation for integrations and extension authors.
