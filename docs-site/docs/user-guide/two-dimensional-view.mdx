# Two-dimensional view {/* #two-dimensional-view */}

The two-dimensional view is the practical workspace for tracing, drawing, and
organizing a composition.

Use it when you need precision and a top-down view of the space.

## Good uses {/* #good-uses */}

The 2D view is useful for:

- placing a floor plan or tracing image;
- drawing walls and room boundaries;
- naming areas such as garage, gate, kitchen, pool, or sidewalk;
- positioning cameras approximately;
- checking whether areas are easy to understand before moving to 3D.

## Start with a tracing image {/* #start-with-a-tracing-image */}

The fastest way to build a first composition is to place an image on the floor
and draw over it.

Useful sources include:

- a floor plan;
- a robot vacuum map screenshot;
- a rough hand drawing;
- a screenshot from a map or layout tool;
- any private reference image that helps you place walls and areas.

The image is a local reference. It does not need to be perfect.

## Keep the first version simple {/* #keep-the-first-version-simple */}

For the first pass:

1. Add only the most important walls.
2. Create broad meaningful areas.
3. Add one or two cameras.
4. Save and verify navigation.
5. Refine after you start using camera views and pipelines.

A useful rough map is better than a detailed model that becomes hard to
maintain.

## Move to 3D when needed {/* #move-to-3d-when-needed */}

After the 2D structure makes sense, use the [3D view](three-dimensional-view.mdx)
to inspect camera placement, models, and spatial context.
