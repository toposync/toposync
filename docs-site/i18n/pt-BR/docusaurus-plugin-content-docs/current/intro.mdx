# Documentação do Toposync

Toposync é uma plataforma local-first para automação residencial, câmeras, contexto espacial e processamento extensível.

## O que existe aqui

- Caminhos de instalação para Python, Docker, Home Assistant e processing servers.
- Documentação de produto para a plataforma principal, extensões e fluxos futuros do app.
- Documentação técnica para integrações e autores de extensões.

## Estado atual

Este site de documentação está sendo inicializado. Os guias de instalação existentes continuam no repositório enquanto a estrutura pública é preparada.
