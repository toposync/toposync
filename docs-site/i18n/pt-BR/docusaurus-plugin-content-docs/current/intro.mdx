# Documentação do Toposync

Toposync é uma plataforma open source, local-first, para **Spatial Home Automation com inteligência local**: câmeras, mapeamento espacial de câmeras, eventos espaciais, composições 2D/3D da casa, integração com Home Assistant e processamento extensível em ambientes residenciais privados.

A versão curta:

- **Spatial Camera Mapping** conecta imagens de câmeras à casa mapeada.
- **Spatial Events** adicionam posição, área, câmera, estado do objeto e contexto aos eventos.
- **Spatial Intelligence** é a camada local mais ampla de interpretação que combina câmeras, áreas, dispositivos, pipelines e histórico.
- **Spatial Awareness** é usado para comportamento de produto sensível a áreas, como reagir de forma diferente perto de portões, piscinas, garagens, calçadas ou entradas.
