# Visualização 2D {/* #two-dimensional-view */}

A visualização 2D é o espaço prático para decalcar, desenhar e organizar uma
composição.

Use quando precisar de precisão e uma visão de cima do espaço.

## Bons usos {/* #good-uses */}

A visualização 2D é útil para:

- posicionar uma planta ou imagem de decalque;
- desenhar paredes e limites de cômodos;
- nomear áreas como garagem, portão, cozinha, piscina ou calçada;
- posicionar câmeras aproximadamente;
- conferir se as áreas são fáceis de entender antes de ir para 3D.

## Comece com uma imagem de decalque {/* #start-with-a-tracing-image */}

A forma mais rápida de construir uma primeira composição é colocar uma imagem no
piso e desenhar por cima dela.

Fontes úteis incluem:

- uma planta baixa;
- screenshot de mapa de robô aspirador;
- um desenho simples feito à mão;
- screenshot de uma ferramenta de mapa ou layout;
- qualquer imagem privada de referência que ajude a posicionar paredes e áreas.

A imagem é uma referência local. Ela não precisa ser perfeita.

## Mantenha a primeira versão simples {/* #keep-the-first-version-simple */}

Na primeira versão:

1. Adicione apenas as paredes mais importantes.
2. Crie áreas amplas e significativas.
3. Adicione uma ou duas câmeras.
4. Salve e valide a navegação.
5. Refine depois que começar a usar câmeras e pipelines.

Um mapa simples e útil é melhor do que um modelo detalhado difícil de manter.

## Vá para 3D quando necessário {/* #move-to-3d-when-needed */}

Depois que a estrutura 2D fizer sentido, use a [visualização 3D](three-dimensional-view.mdx)
para inspecionar posicionamento de câmeras, modelos e contexto espacial.
