# Home Assistant

First-party extension scaffold to configure one or more Home Assistant servers (host + API token) via the global Settings modal.

## Visualizações especiais

No editor do elemento **Home Assistant item**, você pode escolher uma “visualização especial” para um único item:

- **Luminária** (para domínios compatíveis)
- **Vento climatizado** (climate)
- **Modelo 3D**: faz upload de um `.glb/.gltf` e usa o modelo como visual do dispositivo/entidade; quando a entidade está “ligada/ativa”, o modelo anima se tiver animações.
